# ROFAH FASHION CAVE

This is a single-page demo for a fashion design and custom-sew website. It is built with HTML, CSS, and vanilla JavaScript.

## How to Run Locally

To run this project locally, simply open the `index.html` file in your web browser.

## Unsplash Image Credits

The images used in this project are from Unsplash. A complete list of the images, their Unsplash queries, and photographer credits can be found in the `assets/images/unsplash-mapping.txt` file.

To replace an image, download a new one from Unsplash, save it in the `assets/images/` directory with the same filename, and update the `unsplash-mapping.txt` file with the new information.

## Fonts

*   **Headings**: Times New Roman (serif) - Chosen for its classic and elegant feel, suitable for a fashion brand.
*   **Body**: Helvetica (sans-serif) - Chosen for its clean and modern look, providing good readability.
